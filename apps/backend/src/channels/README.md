# Channels Module

Модуль для управления Telegram каналами.

## Возможности

- ✅ Подключение каналов администратором
- ✅ Управление каналами (активация/деактивация)
- ✅ Обновление статистики каналов
- ✅ Получение списка каналов

## API Endpoints

### POST /channels
Подключение канала (только админ)

**Request:**
```json
{
  "channelId": "-1001234567890",
  "channelUsername": "my_channel",
  "channelTitle": "Мой канал",
  "channelType": "channel",
  "isActive": true
}
```

### GET /channels
Получение всех каналов (только админ)

**Query:**
- `includeInactive` - включить неактивные каналы

### GET /channels/:id
Получение одного канала (только админ)

### PUT /channels/:id
Обновление канала (только админ)

### DELETE /channels/:id
Удаление канала (только админ)

## Как получить channelId

1. Добавьте бота в канал как администратора
2. Отправьте любое сообщение в канал
3. Используйте `getUpdates` API или проверьте логи бота
4. Формат: `-1001234567890` (для супергрупп начинается с -100)
